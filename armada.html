Fill in all the data so the AI has something to work with and then press the applicable red "Compute" buttons to let the AI perform an action. This is still an early version of the software and I will amend only the items I receive feedback about. Go to the main page to give your feedback. Some important notes:

  • The software is ideally used in a smartphone browser.
  • The values need to be hand amended (for example, if "Decrease speed" command is given the user needs to change the ship's speed.
  • Command tokens are computed and played once per turn instead of several turns in advance.
  • For the solo ship's target use the nearest enemy capital ship (and stick with it). If there is an objective target, flip a coin. Heads the ship goes for the nearest target, tails the ship goes for the objective target.
  • If you want the software to change/improve give your feedback! Don't be shy. This is the first release so there will still be "gaps" in the rules, please report them as you find them.

    This is a work in progress area, for now, use the following standard rules if the solo ship attacks:

  • Solo ship attacks target's weakest shields available
  • If solo ship rolls "Accuracy" disable target's defense tokens with the following priority: Scatter, Evade, Brace, Redirect

    • Use the following standard rules for defending solo ship's:

  • Use defense tokens with the following priority: Scatter, Evade, Brace, Redirect
  • Only use Evade at a range where it is effective. Cancel/reroll the strongest die.
  • Only use Redirect when there are stronger hull zones adjacent to the target one. Transfer damage to the strongest zone.

    • Obviously you can feel free to fill in the gaps of the software (attacks, target acquisition etc) with your own rules/system.
